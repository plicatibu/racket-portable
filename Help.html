Don't Panic!
Racket has a variety of resources designed to help you with any problems you may have.
Help Desk
Your first stop should always be with the help system that's built into Racket and available from DrRacket's help menu or by pressing F1 with the cursor on a search term. This documentation is customized for your installation, and may include documentation for optional packages you've installed. As a second line of defense, the documentation for the core of the most recent version of Racket is available from this web site (for the installed version there is also a local copy of the documentation).
Not sure what to search for? The documentation includes a guide (also located in your local copy of the documentation) that provides a narrative introduction to many of Racket's features.
Learning how to Program
Try going through How to Design Programs.
Searching the Web
The Schematics Scheme Cookbook is a wiki for Scheme snippets, many of which work in Racket. Additionally, your favorite search engine may well provide answers for many of your questions.
The Mailing List
The users@racket-lang.org mailing list is a great source for answers to questions when the above resources don't pan out; sign up for it in the Community area of the website.

Thanks for using Racket!
